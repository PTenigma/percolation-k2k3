# percolation-embedding
How much data is enough? Training data= translucent epsilon-balls in embedding space colored by cluster size (blue→red). Drag N to watch percolation into a red giant component, then at BBP transition, a new eigenmode K(N) opens and the cover shatters. Based on our papers, [1] https://arxiv.org/abs/2604.17581 and [2] https://arxiv.org/abs/2512.23109
